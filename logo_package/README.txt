TechFlow Logo Package
==================

This package contains the complete TechFlow logo set in multiple formats and sizes.

Contents:
- Vector formats (SVG, EPS, AI)
- Raster formats (PNG, JPG) in multiple resolutions
- Various logo versions (horizontal, vertical, icon, monochrome, white, black)

Usage Guidelines:
- Use SVG for web applications
- Use PNG for digital media with transparency needs
- Use JPG for print media
- Use EPS/AI for professional print and design work

Resolutions Available:
- 64x64, 128x128, 256x256, 512x512 (Icon)
- Standard, 2x, 4x scaling for other versions
